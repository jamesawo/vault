
# Vault

Vault is a simple local-first iOS application for securely storing personal files.

The project is built with SwiftUI and follows a composable architecture using reusable Swift Packages. Features such as authentication, encryption, file import, and secure storage are developed as independent modules that can be reused across future applications.

## Goals

- Local-only storage
- Face ID and device passcode authentication
- Encrypted file storage
- Reusable Swift Packages
- Clean and maintainable architecture
